# Audit 模块：审计与对账

## 1. 模块目标

负责记录关键业务行为，确保返利、提现、配置和管理员操作可追溯。

## 2. 模块边界

本模块负责：

- 操作日志。
- 返利流水。
- 提现流水。
- 配置修改日志。
- 对账导出。

本模块不负责：

- 业务规则计算。
- 支付网关调用。

## 3. 必须审计的行为

- 返利发放。
- 返利取消。
- 提现申请。
- 提现审核。
- 提现打款。
- 配置修改。
- 管理员手动调整余额。
- 邀请关系修正。
