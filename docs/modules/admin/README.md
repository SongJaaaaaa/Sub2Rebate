# Admin 模块：Filament 管理后台

## 1. 模块目标

负责平台运营者管理系统。

## 2. 模块边界

本模块负责：

- 用户管理。
- 邀请关系管理。
- 返利流水。
- 提现审核。
- 配置管理。
- 风控管理。
- 报表查看。

本模块不负责：

- 用户前台。
- 支付宝实际打款逻辑。

## 3. 认证方案

```text
用户数据源：Sub2API 用户表
登录验证：Filament 自带登录，校验 Sub2API 用户表
角色：管理员（admin）
权限标记：Sub2Rebate 维护独立角色表，标记哪些用户是管理员
```

只有被标记为管理员的用户可登录 Filament 后台。

## 4. 后台页面

第一版页面：

- 用户列表。
- 邀请关系。
- 里程碑进度。
- 返利流水。
- 返利余额。
- 提现审核。
- 配置中心。
- 黑名单。
