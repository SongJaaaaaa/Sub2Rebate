# Invite 模块：邀请关系与邀请树

## 1. 模块目标

负责维护用户之间的邀请关系，生成树状裂变结构，并为里程碑奖励和多级返利提供上级链路。

## 2. 模块边界

本模块负责：

- 邀请码。
- 推广链接。
- 直接上级绑定。
- 邀请路径。
- 邀请树查询。
- 防止循环邀请。

本模块不负责：

- 返利金额计算。
- 提现。
- 支付。

## 3. 核心概念

| 概念 | 说明 |
|---|---|
| inviter | 邀请人，直接上级 |
| invitee | 被邀请人，下级 |
| referral path | 从根用户到当前用户的路径 |
| depth | 用户在邀请树中的深度 |

## 4. 建议数据结构

| 表 | 说明 |
|---|---|
| referral_paths | 邀请路径表 |

核心字段：

```text
user_id
parent_id
path
depth
root_inviter_id
invite_code
created_at
updated_at
```

## 5. 核心流程

```text
用户访问推广链接
-> 注册 / 登录
-> 校验邀请码
-> 绑定直接上级
-> 生成 path 和 depth
-> 生成自己的邀请码
```
